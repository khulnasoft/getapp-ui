---
title: Dark Theme
description: Getapp-ui Design has in-built dark theme support. We use Sass (with the SCSS variant) for easier maintenance.
---
import CodePreview from "@components/CodePreview.svelte";

| CSS Class     | SCSS Variable     | Theme         |
|---------------|-------------------|---------------|
| Default       | -                 | Light         |
| `.theme-dark` | `$theme-dark`     | Dark          |

## Global Sass variable
This global CSS class is declared as a Sass variable in order to be used inside partials
<br />

<div class="box u-stretch" style="padding: 1rem; padding-right: 2.5rem;">
    <CodePreview
        client:load
        code={`
    <style language="scss">
        $theme-dark: ".theme-dark";
    </style>
`}
        language="html"
        lineNumbers
    />
</div>

## Implementation to a SCSS Partial

<br />

<div class="box u-stretch" style="padding: 1rem; padding-right: 2.5rem;">
    <CodePreview
        client:load
        code={`
    <style language="scss">
    $theme-dark: ".theme-dark";
    .partial {
       --p-partial-bg-color: var(--color-neutral-0);
       --p-partial-color: var(--color-neutral-60);
       background-color: hsl(var(--partial-bg-color));
       color: hsl(var(--partial-color));
       /* Dark Theme */
       #{$theme-dark} & {
         --p-partial-bg-color: var(--color-neutral-100);
         --p-partial-color: var(--color-neutral-20);
       }
    }
    </style>
`}
        language="html"
        lineNumbers
    />
</div>
