## Breakpoints

In the GetApp console we use three screen ranges (small / medium / large):

| Screen Size    | Range             | Scss Variable     |
| -------------- | ----------------- | ----------------- |
| Small          | 0px - 767px       | `$break1`         |
| Medium         | 768px - 1198px    | `$break2`         |
| Medium (open)  | 768px - Infinite  | `$break2open`     |
| Large (open)   | 1199px - infinite | `$break3open`     |



## CSS Implementation

<br />

<div class="box u-stretch" style="padding: 1rem; padding-right: 2.5rem;">
    <CodePreview
        client:load
        code={`
    <style>
    /* affect all screens sizes */
    .partial {
       --p-padding: 1.25rem;
       padding: var(--p-padding);
    }
    /* affect only small screens */
    @media (max-width:767.99px) {
        .partial {
            --p-padding: 1rem;
        }
    }
    /* affect only small & medium screens */
    @media (max-width:1198.99px) {
        .partial {
            --p-padding: 1rem;
        }
    }
    /* affect only medium and larger screens */
    @media (min-width:768px) {
        .partial {
            --p-padding: 1rem;
        }
    }
    /* affect only small and larger screens */
    @media (max-width:767.99px) and (min-width:1199px){
        .partial {
            --p-padding: 1rem;
        }
    }
    </style>
`}
        language="html"
        lineNumbers
    />
</div>

## Special Use Cases

In some use cases, we might want to affect only a part of screen sizes and not all of them. In Getapp-ui Design responsive library, there is no overlap between the different resolutions. As shown in the example below, you can affect more than one screen size by adding breakpoints.

<br />

<div class="box u-stretch" style="padding: 1rem; padding-right: 2.5rem;">
    <CodePreview
        client:load
        code={`
    <style>
        .partial {
            /* here you put code for all screen sizes */
            @media #{$break1}                   { } /* only small screens */
            @media #{$break1}, #{$break2}       { } /* small & medium screens */
            @media #{$break2}                   { } /* only medium screens */
            @media #{$break1}, #{$break3open}   { } /* small & large screens */
        }
    </style>
`}
        language="html"
        lineNumbers
    />
</div>
